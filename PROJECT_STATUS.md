## 使用说明

### 1. 首次运行
```bash
# 以管理员权限运行
run_as_admin.bat
```

### 2. 登录系统
- 使用管理员账户登录：admin/admin123
- 或使用NFC卡刷卡登录（如果已绑定）

### 3. 用户管理
- 点击"用户管理"按钮
- 需要重新验证管理员身份
- 可以添加、编辑、删除用户

### 4. NFC功能
- 连接NFC读卡器到串口
- 点击"连接NFC"按钮
- 刷卡进行登录或绑定
